DataGenerator.7z contains datasets used with the DataGenerator tool to generate and evolve identity, location, and event data.

This data is only used for the DataGenerator utility, so if you don't plan on using that, you may ignore obtaining these files.

The following data is used
* Person data.  Included. A selection of popular first names by gender, sur names, trades, and traits.
* Geolocation data.  Not included. Select extracts from GeoNames Gazetteer (via GeoNames Gazetteer - http://download.geonames.org/export/dump/)
* WordNet DB 3.1.  Not included. Select extracts from the WordNet DB 3.1 dataset (via https://wordnet.princeton.edu/wordnet/download/current-version/)
