Obtain geolocation data from GeoNames Gazetteer - http://download.geonames.org/export/dump/

This data is only used for the DataGenerator utility, so if you don't plan on using that, you may ignore obtaining these files.


Suggested Files:
admin1CodesASCII.txt
admin2Codes.txt
alternateNames.txt
countryInfo.txt
featureCodes_en.txt
iso-languagecodes.txt
* Desired country data (eg: CA.txt, GB.txt, US.txt)
